# HelloWorld

1. 新建一个JAVA文件

   - 文件后缀名为.java
   - HelloWorld.java

2. 编写代码

   ```java
   public class HelloWorld {
       public static void main(String[] args) {
           System.out.println("Hello World!");
       }
   }
   ```

3. 编译

   ```
   D:\Java\Project\src>javac HelloWorld.java
   
   D:\Java\Project\src>java HelloWorld
   Hello World!
   
   D:\Java\Project\src>
   ```

   

# 注意事项

1. 每个单词的大小写不能出现问题,**Java大小写敏感**;
2. 尽量使用英文目录;
3. 文件名和类名必须保证一致;
